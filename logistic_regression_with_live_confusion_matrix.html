Logistic Regression: Two Views of the Same Problem

The model draws a sigmoid probability curve through binary observations, while optimisation takes place in the weight–bias plane. Change the parameters and watch the probability model, decision boundary, cross-entropy landscape, and gradient path update together.

Binary cross-entropy
J(w,b) = −(1/n) Σ [yᵢ ln pᵢ + (1−yᵢ) ln(1−pᵢ)]
